## Description

一个用于管理python及相关开发的工具

keywords:tool,project_manager

## 特点

+ 根据模板快速构建python,node项目,支持flask,sanic,celery
+ 简单的编译和打包指令,支持wheel,egg,cython,和docker编译
+ Cython支持良好
+ 快速测试
+ 文档维护,支持github page
