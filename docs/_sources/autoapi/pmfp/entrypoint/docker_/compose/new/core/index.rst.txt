.. class:: New

   Bases: :py:obj:`pmfp.utils.endpoint.EndPoint`

   创建一个docker-compose文件.

   当指定的dockercompose文件存在时创建全新内容并覆盖原来老的compose文件,老的会被重新保存为`原名.{timestamp}_bak`;
   当指定的dockercompose文件不存在时创建新的compose文件.
   更新操作只能更新如下内容:

   1. service
   2. 外部networks声明
   3. 外部volumes声明
   4. 外部configs声明
   5. 外部secrits声明
